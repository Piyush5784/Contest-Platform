# Next.js Starter Template

## Tech Stack

- React v19
- Next.js v15

## Environment Variables

```env
NEXTAUTH_URL="http://localhost:3000"
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
NEXTAUTH_SECRET=
DATABASE_URL="postgresql://postgres:mysecpass@localhost:5432/postgres"
```

## Features

- Login page
- Register page
- Account page
- Dashboard page (empty)
- Built with shadcn/ui components
- Authentication: Credentials and Google login
